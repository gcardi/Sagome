# Sagome
Programma pilotaggio sagome per Federazione Sammarinese Tiro a Segno

Necessita di FFTW per Windows (le DLL).

La struttura delle cartelle deve essere la seguente:

```
├───FFTW
│   ├───fftw-3.3-dll32
│   └───fftw-3.3-dll64
├───Libs
│   ├───Win32
|   |     libfftw3-3.lib
│   └───Win64
|         ibfftw3-3.a
└───Risorse
```
